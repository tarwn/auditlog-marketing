Embedding the AuditLog

AuditLog provides a drop-in you can embed in your own application, with customization available for things liek the logo, links, and columns that most make sense for your system.

Customize Screen

Open the Customize screen and access the "Embed Script" tab for an example:

Here is an example of using the script:

    <!DOCTYPE html>
    <html lang="en-US" dir="ltr">
    <head>
        <meta charset="utf-8">
        <title>View your audit log</title>
    </head>
    <body>
        <div id="audit-log"></div>
        <script src="https://app.auditlog.co/dropin/dropin.js"></script>
        <script>
            auditLog({
                view: '{your-view-id}',
                accessKey: '{your-view-access-key}',
                clientId: '{your-client-id}',
                target: 'audit-log',
                host: 'https://app.auditlog.co'
            });
        </script>
    </body>
    </html>

There are 3 key pieces to this:

  • The audit-log div, which will be used to render the Audit Log
  • The dropin.js script which includes all of the code
  • The auditlog({}) options which provide the view id/access key and identify the client to display
